ในนี้ได้ฝึกตัวของ js 
ฝึกการทำงานของตัว String method ว่า return แล้วออกคร่าอย่างไร 
Array method ความแตกต่างรหว่างของตัว String และ Array ว่ามีความแต่กต่างกันยังโดยการเขียนให้เห็นภาพ
ยกตัวอย่างเช่น สร้าง array ของตัวเลขและ string หรือ ค้นหาตำแหน่งของ element
 ความแตกต่าง == กับ === ว่าควรต้องใช้ตัวไหน
 ความแตกต่างระหว่าง string primitive กับ String object
