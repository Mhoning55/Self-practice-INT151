1. Callback Functions (ฟังก์ชันกลับ)
Callback function คือฟังก์ชันที่ถูกส่งเข้าไปเป็นอาร์กิวเมนต์ให้กับฟังก์ชันอื่น เพื่อเรียกใช้งานในภายหลัง


2. การใช้ Array Methods พื้นฐาน
2.1 .filter()
ใช้กรองสมาชิกใน array ตามเงื่อนไขที่กำหนดใน callback function

2.2 .map()
ใช้แปลงข้อมูลใน array เช่น แปลงชื่อเป็นตัวพิมพ์ใหญ่

2.3 .reduce()
ใช้รวบรวมค่าจากสมาชิกใน array เช่น รวมราคาสินค้าทั้งหมด

2.4 .push() และ .unshift()
.push() เพิ่มสมาชิกที่ท้าย array และคืนค่าความยาวใหม่ของ array
.unshift() เพิ่มสมาชิกที่ต้น array

2.5 .pop() และ .shift()
.pop() ลบสมาชิกตัวสุดท้ายใน array
.shift() ลบสมาชิกตัวแรกใน array

2.6 .slice()
ใช้ตัดหรือเลือกสมาชิกบางส่วนจาก array โดยไม่แก้ไข array เดิม
รองรับทั้งการใช้ index บวกและลบ

3. การจัดการข้อมูลในรูปแบบ Object และ Array
ใช้ .map() เพื่อดึงและแปลงข้อมูลในอ็อบเจกต์ เช่น แปลงชื่อพนักงานให้เป็นตัวพิมพ์ใหญ่และรวมชื่อกับนามสกุล
ใช้ .filter() ร่วมกับ .map() เพื่อกรองชื่อที่มีคำเฉพาะ เช่น คำว่า "JAI"

4. การคำนวณและใช้งานกับข้อมูลเชิงตัวเลข
ใช้ .reduce() เพื่อหาผลรวมของราคาสินค้าในตะกร้า (cart) โดยคูณราคาต่อหน่วยกับจำนวนสินค้าแต่ละชนิด
คำนวณ VAT 7% จากราคาสุทธิที่รวมแล้ว
ตัวอย่าง:

const totalPrice = carts.reduce((acc, cart) => acc + cart.price * cart.amount, 0);
const netPrice = totalPrice * 1.07; // เพิ่ม VAT 7%
