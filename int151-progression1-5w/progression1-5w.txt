จากที่ได้ลองฝึกเขียน JavaScript มาหลายหัวข้อ รู้สึกว่าได้เข้าใจพื้นฐานของภาษาเยอะขึ้นมาก 
โดยเฉพาะเรื่องการจัดการกับ string กับ array ซึ่งเป็นสิ่งที่ใช้งานบ่อยสุด ๆ 
ได้ฝึกใช้เมธอดต่าง ๆ อย่าง .split(), .toLowerCase(), .filter(), .map(), .reduce() 
แล้วก็เห็นชัดเลยว่าการทำงานกับข้อมูลมันมีหลายวิธีขึ้นอยู่กับว่าต้องการผลลัพธ์แบบไหน
นอกจากนี้ก็ได้รู้ความต่างระหว่าง string กับ array ว่าแม้จะดูคล้าย ๆ กันเวลาเข้าถึง 
index หรือใช้ .length แต่ก็มีเมธอดที่ใช้งานต่างกัน อย่าง array จะสามารถเพิ่ม 
ลบ คัดกรองสมาชิกได้หลายวิธี 
ซึ่ง string จะเปลี่ยนค่าไม่ได้โดยตรง ต้องแปลงหรือใช้เมธอดช่วย
ได้เรียนรู้เรื่องฟังก์ชันด้วย ทั้งการเขียนแบบปกติ (function declaration) 
กับแบบ expression รวมถึงเรื่อง callback function 
ที่เป็นพื้นฐานสำคัญเวลาเขียนโค้ดแบบ async หรือโค้ดที่ต้องส่งฟังก์ชันไปให้ฟังก์ชันอื่นเรียกต่อ
อีกจุดที่ได้จากการฝึกคือเรื่อง object ที่ได้ลองทั้งการเพิ่ม property, 
รวม object เข้าด้วยกัน แล้วก็ destructuring 
ข้อมูลออกมา รวมถึงการนับคำในประโยคผ่าน object ทำให้เห็นว่า object 
เป็นเครื่องมือที่ดีในการจัดการข้อมูลที่เป็น key-value
โจทย์ที่ฝึกยังมีเรื่องตัวเลข เช่น หาค่ามากสุด น้อยสุด ค่าเฉลี่ย หรือเลือกเฉพาะเลขคู่ 
เลขคี่ แล้วแปลงค่าต่าง ๆ ด้วยเมธอดของ array ซึ่งทำให้มองภาพการจัดการข้อมูลในชีวิตจริงได้ง่ายขึ้น
