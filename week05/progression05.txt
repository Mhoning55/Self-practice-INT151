เราจะได้ฝึกเพิ่ม property เข้าไปใน object, รวม object เข้าด้วยกัน, 
นับจำนวนคำในประโยคแบบไม่สนใจตัวเล็กตัวใหญ่ แล้วก็ลองเปลี่ยนชื่อ property เวลา destructuring ข้อมูลออกมา 
อีกส่วนก็มีการรวม object แล้วดึง property ที่ต้องการออกมา
ในเรื่อง string ก็มีการลบคำที่ซ้ำออกแต่ยังเก็บคำแรกไว้ ฝึกคิดเรื่อง set และ filter ส่วน array 
ก็มีโจทย์ให้หาค่า min, max, ผลรวม และค่าเฉลี่ย รวมถึงการแปลงข้อความให้เป็นตัวพิมพ์ใหญ่ทั้งหมด 
การหาค่ามากที่สุดด้วย reduce การลบค่า falsy (พวก 0, null, undefined, false, '')
ยังมีโจทย์ให้จัดการตัวเลข เช่น เลือกเอาเฉพาะเลขคู่แล้วคูณสอง หรือเลือกเฉพาะเลขคู่กลับมาเฉยๆ, 
สร้าง array ที่เต็มไปด้วยค่าที่กำหนดตามความยาว, การยกกำลังสองให้กับทุกตัว 
และโจทย์สุดท้ายคือเอาเลขทั้งหมดมายกกำลังสองก่อน แล้วคัดเอาเฉพาะเลขคี่ออกมา จากนั้นรวมผลรวมให้ได้คำตอบสุดท้าย
